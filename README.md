## ReNotes

A zettle-kasten based markdown note-taking that you can selfhost.

### Features

- [x] Collections
- [x] Live Markdown Editing
- [x] Authentication
- [ ] Public collections
- [ ] Deep Searching
- [ ] Graph

### Self Host

```sh
docker build . -t renotes/renotes
docker-compose up -d
```
